# Bolivia service boundary

`bolivia_admin0_ne10m.geojson` contiene unicamente el poligono de Bolivia
extraido de Natural Earth, **Admin 0 - Countries**, escala 1:10m.

- Fuente: https://www.naturalearthdata.com/downloads/10m-cultural-vectors/10m-admin-0-countries/
- Repositorio: https://github.com/nvkelso/natural-earth-vector
- Commit: `ca96624a56bd078437bca8184e78163e5039ad19`
- Licencia: dominio publico, segun Natural Earth.
- Fecha de extraccion: 2026-07-10.

Las coordenadas conservan el orden GeoJSON `[longitud, latitud]`. El backend
considera validos tanto el interior como el borde del poligono.
